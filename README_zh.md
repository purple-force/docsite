# Docsite

docsite 是一款开源站点搭建工具，依据其提供的功能，能够快速搭建一个开源站点。

## 特性

- 支持文档、博客、项目首页一键生成
- 支持markdown文件
- 跨屏幕适配的 UI
- SEO 优化
- 国际化
- 支持自定义页面

## 依赖

- node.js > 6.x

## 快速开始

1、Install

```
$ npm i -g docsite
```

2、Init a project

```
$ cd my-project
$ docsite init

# or
$ docsite init my-project
```

3、Local preview


```
$ docsite start
```

4、Build

```
$ docsite build
```
